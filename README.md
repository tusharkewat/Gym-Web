🏋️ Gym Management Website

A modern and responsive Gym / Fitness Website built using HTML, CSS, and JavaScript.
This project provides a clean user interface for showcasing gym services, classes, trainers, and membership plans.

🔗 Live Demo:
https://tusharkewat.github.io/Gym-Web/

📌 Project Overview

This project is designed to create an attractive fitness website landing page where users can explore gym services, training programs, class schedules, and pricing plans.

The website focuses on:

Clean UI/UX

Responsive design

Simple navigation

Fitness service showcase

It can be used as:

Gym landing page

Fitness trainer website

Gym membership promotion website

Frontend practice project

🚀 Features

✅ Responsive Design (Mobile Friendly)
✅ Modern Gym Landing Page
✅ Navigation Menu
✅ About Gym Section
✅ Services / Classes Section
✅ Workout Schedule
✅ Pricing Plans
✅ Contact Section
✅ Smooth UI Experience

🛠️ Technologies Used

HTML5 – Website structure

CSS3 – Styling and layout

JavaScript – Interactivity

Users visit the homepage.
